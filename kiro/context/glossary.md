# Fur & Co Glossary

## Business Terms

**Tri-State Ecosystem**: The core UX concept where the application adapts its entire theme and experience based on three distinct modes (Gateway, Core, Origin).

**Unlimited**: Premium subscription service focused on recurring pet care products. Associated with the CORE theme state.

**Niche**: Artisanal, story-driven product line emphasizing organic and handcrafted items. Associated with the ORIGIN theme state.

**Gateway**: The default entry experience - neutral and accessible. The standard e-commerce experience.

## Technical Terms

**Theme Context**: React Context managing the global `currentMode` state that controls which of the three themes is active.

**Data Theme Attribute**: HTML `data-theme` attribute applied to the root element to scope CSS variables for theming.

**MCP**: Model Context Protocol - used for AI assistant integration and context management.

## Product Categories
- Walk (leashes, harnesses, collars)
- Dine (bowls, feeding accessories)
- Rest (beds, blankets)
- Play (toys, entertainment)
- Health (supplements, medications)
- Care (grooming, hygiene)
- Gear (carriers, travel accessories)
- Treats (food, snacks)

## User Journey States
- **Anonymous**: Browsing without account
- **Authenticated**: Logged in user
- **Subscriber**: User with active Unlimited subscription
- **Referrer**: User participating in referral program
